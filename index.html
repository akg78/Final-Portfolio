Project Brief - Analog Clock HTML/CSS/JS Website

Project Overview
Create a visually appealing and interactive analog clock website using HTML and CSS. The website will display a traditional analog clock face with hour, minute, and second hands, along with a digital readout of the current time.
Design and Layout
The clock face will have a classic design with Roman numerals representing the hours. The clock hands (hour, minute, and second) will move smoothly to indicate the current time. The website's layout will be clean and user-friendly, with the clock centered on the page.
    Technical Stack
  • HTML/CSS Version HTML5 and CSS3
  • JavaScript: Used for clock movement and time calculations.
